Word Radar

Click-to-lookup Chrome Extension — API Server

● Server Online

Endpoints

GET /health
Health check. Returns {"status":"ok"} when the server is running.
GET /api/lookup?q={word}&context={context}
Look up a word. Returns phonetic, definitions, examples, and audio URL. If context is provided, the word is automatically saved to the word list.
q (required) — the word to look up
context (optional) — sentence context for the word
GET /api/wordcard?q={word}
Generate a rich word card with LLM-enhanced etymology, mnemonics, and cognates. Falls back to dict-only data when LLM is not configured.
q (required) — the word to generate a card for
GET /api/words?limit={{.Limit}}&offset={{.Offset}}
List saved word records with pagination.
limit — max records (default 20, max 100)
offset — pagination offset (default 0)
GET /api/words/lookups?start={start}&end={end}&word={word}
Query word lookup frequency within a time range. Queries the word_lookups table. Returns per-word: range count, total count, first and last lookup time, sorted by range count descending.
start (required) — range start, RFC3339 format (2026-01-01T00:00:00Z)
end (required) — range end, RFC3339 format (2026-06-01T23:59:59Z)
word (optional) — filter to a specific word; omit for all words

Examples

$ curl http://localhost:{{.Port}}/api/lookup?q=serendipity
{
  "word": "serendipity",
  "phonetic": "/ˌsɛrənˈdɪpɪti/",
  "meanings": [...],
  "examples": [...]
}
$ curl "http://localhost:{{.Port}}/api/words/lookups?start=2026-01-01T00:00:00Z&end=2026-06-01T23:59:59Z"
[
  {
    "word": "mandarin",
    "range_count": 4,
    "total_count": 10,
    "first_lookup_at": "2026-05-30T10:15:00Z",
    "last_lookup_at": "2026-05-31T07:59:29Z"
  }
]
